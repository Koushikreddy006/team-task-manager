# Team Task Manager (Flask)

## 🔥 Features
- User Signup & Login
- Add tasks
- Mark tasks as Done
- Delete tasks
- Dashboard UI

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS (Bootstrap)

## ▶️ Run Locally
pip install -r requirements.txt  
python app.py
