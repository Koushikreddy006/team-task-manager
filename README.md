Team Task Manager (Full-Stack Web Application)

Live Application:
https://web-production-2ff3e4.up.railway.app

GitHub Repository:
https://github.com/Koushikreddy006/team-task-manager

Description:
This is a full-stack web application that allows users to manage projects and tasks with role-based access control.

Features:
- User Authentication (Signup/Login)
- Role-based access (Admin & Member)
- Admin can create projects
- Admin can assign tasks to users
- Members can view assigned tasks
- Task status tracking (Pending / Done)
- Delete and manage tasks
- Dashboard view for all activities

Tech Stack:
- Python (Flask)
- SQLAlchemy (ORM)
- SQLite (Database)
- HTML, CSS, Bootstrap (Frontend)
- Railway (Deployment)

How to Use:
1. Signup as Admin and Member
2. Login as Admin
3. Create a project
4. Assign tasks to users
5. Login as Member to view and update tasks

Note:
The application is deployed on Railway and fully functional online.
