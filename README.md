# Team Task Manager (Flask)

## 🔥 Features
- User Signup & Login
- Role-based access (Admin / Member)
- Add tasks
- Mark tasks as Done
- Delete tasks
- Dashboard with Bootstrap UI

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS (Bootstrap)
- Flask-Login
- SQLAlchemy

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python app.py
